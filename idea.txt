Nature of purchases are expensive but rewarding with multiple tiers to their level and a special final tier.

Assign costs in code.

GUI can be accessed from the facility but upgrades are only available while in transit.

Networking Strategy:
All clients and host pull current upgrades from train name and set
local vars to it
All clients and host have up to date upgrades on their GUI
All clients and host can purchase upgrades on their GUI
If they do:
- update own upgrade var number and set train name to updated var number
- all clients and host read new train name and set vars
- only hosts mod will apply changes to all players via IsOwner on read

Ensures:
- All players can interact with the mod menu
- All players see live updates to their current upgrades
- Only hosts instance applies changes to not stack changes
